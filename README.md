# Тренажёр: Промышленная безопасность

Интерактивный тренажёр для подготовки к проверке знаний по ФЗ-116 «О промышленной безопасности опасных производственных объектов» и сопутствующим нормативным актам.

- 211 вопросов с вариантами ответов (включая вопросы с несколькими правильными ответами)
- Случайный порядок при каждом запуске
- Фильтры: все вопросы / с одним правильным / с несколькими правильными
- Подсветка правильного и неправильного ответа, итоговый результат с процентом и временем
- Режим «Разобрать ошибки» — повторное прохождение только тех вопросов, где была допущена ошибка
- Полностью офлайн после первой загрузки

## Локальный запуск

```bash
bun install
bun run dev
```

Откроется на http://localhost:5173

## Сборка

```bash
bun run build
```

Готовая статика окажется в `dist/`. Перед деплоем убедись, что `dist/data/prombez/questions.json` существует (он лежит в репозитории и не пересобирается).

## Деплой на GitHub Pages

1. Собери проект: `bun run build`
2. Закоммить содержимое `dist/` (или используй GitHub Actions)
3. В настройках репозитория → Pages → Source: branch `main`, folder `/dist`

## Деплой на любой статический хостинг

Можно закинуть содержимое `dist/` на Netlify, Vercel, Cloudflare Pages, Яндекс.Cloud Object Storage — приложение это просто набор статических файлов.

## Источник

Тест по промышленной безопасности ОПО (ФЗ-116 и смежные нормативные акты Ростехнадзора).
